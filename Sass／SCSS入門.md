# Sass の基礎学習

## 概要  
CSS を効率的に書くための記法で２つの記法（拡張子）がある。 
* .sass（Ruby/Haml ベース）
* .scss（CSS３ ベース）

`sass` は Ruby ベースでデザイナーにとって分かりにくい記法であったため、  
現在は、`scss` の方が主流である。

## 公式サイト  
https://sass-lang.com/  

## インストール
```bash
# scss には ruby が必要なので、インストールしておくこと
ruby -v
sudo gem update --system
sudo gem install sass
```
