# 初期インストール

```bash
npm install -g yarn
yarn create react-app {アプリケーション名} --template typescript
```

# アプリ起動

```bash
yarn start
```

# sass インストール

```bash
yarn add node-sass
```

# Material-UI インストール

```bash
yarn add @material-ui/core
yarn add @material-ui/icons
```
